Database characteristics are stored in the db.opt file in the database directory. The CHARACTER SET clause specifies the default database character set. The COLLATE clause specifies the default database collation. Chapter 10, Character Sets, Collations, Unicode, discusses character set and collation names.13.1.12 CREATE EVENT Syntax13.1.13 CREATE FUNCTION Syntax13.1.14 CREATE INDEX Syntax13.1.15 CREATE
